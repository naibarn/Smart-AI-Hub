# FR-1: Multi-method Authentication

## Requirements
- Google OAuth 2.0 integration
- Email/password with BCRYPT (cost factor: 12)
- Email verification (6-digit OTP, 15-min expiry)

## Acceptance Criteria
- Registration completion rate > 70%
- Login success rate > 99%
